## 🛰️ O Manifesto
O **Hyperion** não é apenas um chat. É uma **infraestrutura de soberania digital**. Em um mundo onde as grandes corporações ditam como você deve se comunicar, nós entregamos o código para você construir seu próprio reino.

### 🧩 Por que ser um Arquiteto Hyperion?
* **🌐 Soberania Total (Self-Hosted):** Hospede em sua TV Box, PC ou Raspberry Pi. O servidor é seu, os dados são seus.
* **🎨 Front-end Camaleão:** Use HTML, CSS, JS, TS e PHP. Se quiser que o chat pareça o terminal do Matrix ou um PS2, você decide.
* **⚡ Performance Ninja:** Core em C++ otimizado para hardware limitado. Velocidade bruta sem firulas.
* **🛡️ Watch Dogs System:** Segurança ativa pré-configurada. Proteção contra ataques e moderação automática de elite.

## 🏗️ Guia do Construtor (Quick Start)

### 1. Preparando o Terreno
Certifique-se de ter o **Ninja** instalado e no seu PATH.

### 2. O Ritual de Compilação
```bash
# Clone o Protocolo
git clone [https://github.com/rip-pky/hyperion-chat_doc.git](https://github.com/rip-pky/hyperion-chat_doc.git)
cd hyperion-chat_doc

# Inicie o Ninja
cmake -G Ninja -B build
cmake --build build
