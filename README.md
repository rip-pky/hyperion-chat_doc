# 🌌 Hyperion Chat Protocol
> **"Seu servidor, suas regras, seu código. A liberdade de hospedar o seu próprio reino."**

O **Hyperion** não é apenas mais um aplicativo de chat. É uma infraestrutura de comunicação descentralizada feita para quem cansou das "prisões" corporativas como o Discord. Aqui, a privacidade não é um ajuste, é o padrão. 

Se você quer privacidade real, você hospeda o seu servidor. Simples assim.

## 🚀 Diferenciais do Hyperion

* **Self-Hosting Total (Hospedagem Própria):** O poder de volta para as suas mãos. Rode o servidor na sua TV Box, num PC antigo ou em qualquer lugar que suporte C++.
* **Front-end 100% Programável:** O dono do servidor manda na interface. Use HTML, CSS, JavaScript ou TypeScript para criar um chat que é a sua cara.
* **Performance de Elite:** Core desenvolvido em **C++** e compilado via **Ninja** para garantir velocidade máxima em hardware limitado.
* **Privacidade Radical:** Comunicação direta. O "servidor central" é apenas um indexador; as suas conversas não passam por nós.
* **Customização de Lógica:** Mude o comportamento de teclas (como usar `Espaço` para enviar), crie menus de contexto próprios e defina como a sua comunidade funciona.

## 🥷 Operação Ninja (Build do Sistema)

Para garantir que o Hyperion rode liso, utilizamos o sistema de build **Ninja**.

### Pré-requisitos
* [Ninja Build System](https://github.com/ninja-build/ninja/releases) (Adicionado ao seu PATH)
* Compilador C++ (MSVC, GCC ou Clang)
* CMake 3.10+
